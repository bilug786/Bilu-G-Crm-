# Ex-Employee v.01

**Ex-Employee v.01** is a production-ready, enterprise-grade SaaS CRM designed specifically for travel agencies. Developed by **Bilu G**, it features a clean architecture, modern glassmorphism UI, and robust role-based access control.

## 🚀 Key Features

- **Dashboard**: Real-time analytics, revenue tracking, and booking trends.
- **CRM**: Lead and Customer management with status tracking and follow-ups.
- **Tour Management**: Comprehensive Tour Package, Hotel, and Transport management.
- **Itinerary Builder**: Drag-and-drop day-wise travel plan designer.
- **PDF Generation**: Professional Quotations, GST Invoices, and Vouchers.
- **Financials**: Income/Expense tracking and Sales reporting.
- **Settings**: Fully customizable company branding, banking, and document prefixes.

## 🛠 Tech Stack

- **Framework**: Next.js 15 (App Router)
- **UI**: React 19, Tailwind CSS, shadcn/ui, Framer Motion
- **State/Data**: Zustand, TanStack Query
- **Backend**: Supabase (Auth, Database, Storage)
- **ORM**: Prisma
- **PDF**: @react-pdf/renderer, jsPDF
- **Charts**: Recharts
